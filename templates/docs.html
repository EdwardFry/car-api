GET ENDPOINTS

  • GET /car/id

    • Returns json response containing car with id provided in URL Example

    Example response 
    
                {
                    "data": {
                      "build_date": "25/09/2011", 
                      "colour_id": 1, 
                      "id": 0, 
                      "make": "Honda", 
                      "model": "Jazz"
                    }, 
                    "status": "ok"
                }

POST Requests

  • POST /cars

    • Sends a list of car objects to store in the API. Each car object must contain a "make", "model", "build_date", and valid "colour_id".

    Example body of POST request 
    
            {
                "cars": [
                    {
                        "make": "hondaaaaaaaaaaaaaaaaaa",
                        "model": "jazz",
                        "build_date": "22/06/2017",
                        "colour_id": "2"
                    },
                    {
                        "make": "hondaarrrrrr",
                        "model": "jazz",
                        "build_date": "22/06/2017",
                        "colour_id": "3"
                    }
                ]
            }

  • POST /colours

    • Sends a list of colour objects to store in the API. Each colour object must contain a "name".

    Example body of POST request 
    
            {
                "colours": [
                    {
                        "name": "orange"
                    },
                    {
                        "name": "violet"
                    }
                ]
            }

DELETE Requests

  • DELETE /car/id

    • Deletes car in API with id provided in url.

  • DELETE /colour/id

    • Deletes ccolour in API with id provided in url.
